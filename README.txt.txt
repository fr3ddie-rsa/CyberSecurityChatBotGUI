CyberSecurityChatBotGUI

A Windows Forms chatbot that provides cybersecurity tips.

Features:
- Keyword-based responses (password, scam, privacy, phishing)
- Memory for name and topic
- ASCII art banner
- greeting sound

Usage:
Type keywords like "password", "scam", "privacy", or "phishing" to get tips.
You can also say "my name is ..." or "I'm interested in ..." to personalize responses.
